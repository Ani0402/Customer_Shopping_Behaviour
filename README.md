# 📊 Customer Shopping Behavior

## 📌 Overview

This project demonstrates a complete **data analytics workflow**, starting from raw data ingestion to building an interactive dashboard and generating actionable insights. It showcases skills in **Python, SQL, data cleaning, visualization, and reporting**, making it highly relevant for data analyst roles.

---

## 🛠️ Tech Stack

* **Programming:** Python
* **Libraries:** Pandas, NumPy
* **Database:** MySQL 
* **Visualization:** Power BI

---

## 🔄 Project Workflow

### 1. Data Loading

* Imported dataset from CSV/Excel into Python
* Loaded structured data into SQL database for querying

### 2. Data Cleaning

* Handled missing values, duplicates, and inconsistencies
* Standardized formats (dates, categories, etc.)
* Performed feature engineering for better analysis

### 3. SQL Analysis

* Wrote queries for:

  * Aggregations (SUM, AVG, COUNT)
  * Joins across multiple tables
  * Trend and category-level analysis
* Extracted insights directly from the database

### 4. Data Visualization (Power BI)

* Built an interactive dashboard with:

  * KPI cards
  * Trend charts
  * Category-wise performance breakdown
* Enabled filters and slicers for dynamic exploration

---

## 📈 Key Results

* Identified patterns and trends in the dataset
* Highlighted areas of improvement and growth opportunities
* Provided data-driven recommendations for decision-making
