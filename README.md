# 📊 Data Analytics Project – End-to-End Pipeline

## 📌 Overview

This project demonstrates a complete **data analytics workflow**, starting from raw data ingestion to building an interactive dashboard and generating actionable insights. It showcases skills in **Python, SQL, data cleaning, visualization, and reporting**, making it highly relevant for data analyst roles.

---

## 🛠️ Tech Stack

* **Programming:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
* **Database:** PostgreSQL / MySQL / SQL Server
* **Visualization:** Power BI
* **Reporting & Presentation:** Gamma

---

## 🔄 Project Workflow

### 1. Data Loading

* Imported dataset from CSV/Excel into Python
* Loaded structured data into SQL database for querying

### 2. Data Cleaning

* Handled missing values, duplicates, and inconsistencies
* Standardized formats (dates, categories, etc.)
* Performed feature engineering for better analysis

### 3. SQL Analysis

* Wrote queries for:

  * Aggregations (SUM, AVG, COUNT)
  * Joins across multiple tables
  * Trend and category-level analysis
* Extracted insights directly from the database

### 4. Data Visualization (Power BI)

* Built an interactive dashboard with:

  * KPI cards
  * Trend charts
  * Category-wise performance breakdown
* Enabled filters and slicers for dynamic exploration

### 5. Reporting

* Summarized key insights and business recommendations
* Created a presentation using Gamma for storytelling

## 🚀 How to Run

### 1. Clone the Repository

```bash
git clone <your-repo-link>
cd <project-folder>
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run Python Scripts

```bash
python data_cleaning.py
python analysis.py
```

### 4. Setup Database

* Import dataset into PostgreSQL/MySQL/SQL Server
* Run SQL scripts from `/sql` folder

### 5. View Dashboard

* Open the Power BI `.pbix` file
* Refresh data connections if needed
